The step of setup django web

By John Shi

Published 2016-01-07

Contents


## 1, start a project

$ django-admin.py startproject mysite



## 2, start a app

$ python manage.py startapp polls



## 3, modify ENGINE, NAME

from os import path

PROJECT_ROOT = path.dirname(path.abspath(path.dirname(file)))

DATABASES = {

‘default’: {

‘ENGINE’: ‘django.db.backends.sqlite3‘, # Add ‘postgresql_psycopg2’, ‘mysql’, ‘sqlite3’ or ‘oracle’.

‘NAME’: path.join(PROJECT_ROOT, ‘db.sqlite3’), # Or path to database file if using sqlite3.

‘USER’: ‘’, # Not used with sqlite3.

‘PASSWORD’: ‘’, # Not used with sqlite3.

‘HOST’: ‘’, # Set to empty string for localhost. Not used with sqlite3.

‘PORT’: ‘’, # Set to empty string for default. Not used with sqlite3.

}

}

4, Apply the DB

$ python manage.py migrate

如果 app 里面的 model 有变动: 需要 先执行 makemigrations



5， Enable admin

6, 重新定义 admin 对象的 样式

create a model admin object, then pass it as the second argument to admin.site.register() – any time you need to change the admin options for an object.
